La World Wide Web (WWW):
El desarrollo del frontend moderno está intrínsecamente ligado a la creación de la World Wide Web. Como dato inicial La web y el Internet no son lo mismo. Internet es la infraestructura de redes, mientras que la WWW es el sistema de páginas y enlaces que usamos a diario.

Fue inventada por Tim Berners-Lee en 1989 mientras trabajaba en el CERN (Organización Europea para la Investigación Nuclear) en Suiza.

Teniendo un problema a resolver Tim Berners-Lee buscaba una forma eficiente de compartir información entre científicos de todo el mundo, asi que presentó la propuesta de un sistema basado en hipertexto que permitiera enlazar documentos y acceder a ellos desde cualquier computadora conectada a la red.

Ya como productos:
- En 1990 Creó el primer navegador web y servidor web.
- En 1991: Lanzó el primer sitio web (info.cern.ch).

Expansión global: En 1993, el NCSA Mosaic fue el primer navegador gráfico popular, lo que llevó a la rápida adopción de la web.

Estándares abiertos: En 1994, Berners-Lee fundó el W3C (World Wide Web Consortium) para desarrollar y mantener estándares abiertos de la web.
